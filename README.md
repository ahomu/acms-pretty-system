##Usage

J( 'ｰ`)し themesディレクトリに入れて、sysprettyを継承すればいいのよ...

theme: yourtheme@syspretty

+  a-blog cms v1.5.0以降が必要です
+  上記では便宜上sysprettyとしてますが、テーマ名はなんでもいいです
